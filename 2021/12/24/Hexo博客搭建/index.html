基于Hexo在Windows下搭建博客

前言：属于是电脑小白，就是那种打字二指禅的那种。一共花费了前前后后总共三天时间，之所以写这篇博客主要目的是为了回顾搭建过程以及顺便练习打字速度。这里说一句，万事开头难但请不要放弃，真正的爱好是可以无畏的攀登它的高峰，而不是感叹于它山顶的风景。

必要的准备：

其中前两个是需要下载的，只需要直接在官网下载就行，后面两个是需要在命令里进行配置的

  • Node.JS

  • Git

  • hexo

  • Github

安装说明：

Node.JS安装：

  • 官网下载就行

  • 剩下的就是直接选择安装位置，这里建议安装在其他盘，因为它默认的是C盘

  • 下面要注意的是在选择的时候有的说是选第一项运行环境，有的说是选最后一项添加到环境变量中，虽然我两个都成功了但还是建议直接NEXT就完了，比问我为啥，我也不明白

  • 剩下的直接next,对了其中这项点不点都行，这是询问你是否安装它带的小程序的

    Git安装：直接next就行

检验：

在安装完之后我们需要去验证是否安装成功，看是否是自己下载的版本的话只需要直接启动命令窗口就行，注意这里建议是用管理员模式，因为我在后续的搭建中许多此碰见在用户模式下报错但在管理员模式下成功，所以这里直接建议一开始就直接采用管理员模式执行代码。

  • Git 检查
